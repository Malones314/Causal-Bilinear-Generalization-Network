Download the .pth file from the “trained models” and import the file path into loadCheckpointTest.py to view the test results. The dataset used are MIMII DG and MIMII, which can be downloaded here: https://zenodo.org/records/6529888 and https://zenodo.org/record/3384388 or https://drive.google.com/drive/folders/17Kz4HvcyOePa35TeIww1ZQHgzvdyPY7Q?usp=sharing (Preprocessing has been completed).

The reproduction codes for the comparative methods in the experimental section are stored in: otherMethods.
